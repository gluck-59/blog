# Blog
- Стяните проект, поднимите докер `docker-compose up -d --build`, запустите `composer update`
- Зайдите на главную, запустите сидер по ссылке. Он сходит в интернет, стянет тексты и фотки и зальет все это в базу 
- Если сидер смог достучаться до внешних API, он редиректит вас на главную. Если нет — повторите сидинг

*При сидинге внешние API могут подтормаживать.*
