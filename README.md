# Blog
- Стяните проект, поднимите докер, зайдите на главную
- В первый запуск запустится сидер, сходит в интернет, стянет тексты и фотки и зальет все это в базу 
- Если сидер смог достучаться до внешних API, он редиректит вас на главную. Если нет — повторите сидинг

*При сидинге внешние API могут подтормаживать.*
